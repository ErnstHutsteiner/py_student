
Erstellen Sie zunächst folgende Funktionen

1. Funktion die eine Datei öffnet, 
   return = File
   
2. Funktion die die Zeilen und Zeilenzahl eines Files zurückgibt
   return = Zeilen, Zeilenzahl

3. Funktion die ein neue Datei erstellt
   - sollte die Datei existieren, so wird sie gelöscht
   - erstellt eine neue Datei
	return True oder False
	
4. Funktion die eine Zeile an eine existierende Datei anhängt und diese danach wieder schließt
   return True oder Fals
   
Verwenden Sie nun Ihre Funktionen:   
- Öffnen Sie die Datei Module.txt
- lesen Sie die Zeilen und Zeilenzahl aus
- erstellen Sie eine neue Datei
- hängen Sie jede zweite zuvor gelesene Zeile mit Append an die neue Datei an
